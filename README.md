## **About Project**
현대 IT&E 채용확정형 교육을 들으면서 일주일동안 진행한 Spring Web MVC Framework을 활용한 더한섬닷컴 홈페이지 제작 프로젝트 입니다.<br />
DB 내에 회원 테이블, 상품 테이블, 주문 테이블을 생성한 뒤 Spring Framework와 Oracle DB를 연동하여 회원, 상품, 주문 관련 기능들을 직접 구현했습니다.
